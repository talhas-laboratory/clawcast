# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities to:

- Email: talhaslaboratory@gmail.com

Include:
- affected version
- reproduction steps
- impact summary

Do not post zero-day details publicly before coordinated disclosure.


## Static Analysis Notes

Install-time static scanners may flag generic patterns heuristically.
For ClawCast, review flagged locations and verify no credential forwarding logic exists before restart.
